# Data

This folder contains the raw dataset used for the Profit Leak Analysis project.

## Dataset

Dataset: Global Superstore Data

The Excel file contains three sheets:

- Orders
- Returns
- People

## Main Variables

The Orders sheet includes sales, profit, discount, product, customer, date and region information.

The Returns sheet identifies which orders were returned.

The People sheet contains regional manager information.

## Use in This Project

This data will be used to analyse:

- Sales performance
- Profit margins
- Discount impact
- Product profitability
- Regional performance
- Return patterns
